# Requirements
- `ncurses` library
- - `sudo apt-get install libncurses5-dev libncursesw5-dev`
- `g++` compiler
- - `sudo apt-get install g++`

# How to compile & run
## Compile
- `g++ -o run main.cpp -lncurses -lmenu -std=c++17`

## Run
- `./a.out`

# Machine

- VMWare Workstation 16 Player
- Ubuntu 64-bit 22.04.4-desktop
- 2 CPU cores
- 12 GB RAM
